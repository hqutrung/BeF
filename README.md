# Ứng dụng BeFriend

**BeFriend** là một ứng dụng chia sẻ ảnh miễn phí cho phép người dùng chụp ảnh trên điện thoại của mình sau đó chia sẻ cho bạn bè trên chính mạng xã hội **BeFriend**.
